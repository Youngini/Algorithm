# [BOJ] 1932번: 정수 삼각형

## 📝풀이

2차원 배열을 1차원 배열이라고 생각하고 풀이를 진행했다.
예를 들어 3층 정수 삼각형을 만드는 경우,
    1
   2 3
  4 5 6

이 형태가 2차원 배열에 아래와 같이 들어가있다고 가정하고,

1 0 0 0
2 3 0 0
4 5 6 0

이를 1차원 배열로 바꿨다.단, 인덱스 0번은 비워두고 1번부터 시작했다. (line5~line8)

1 0 0 0 2 3 0 0 4 5 6 0

이렇게 하면 array[j] 의 왼쪽 아래에 있는 것은 array[j+n] 이 되고 오른쪽 아래에 있는 것은 array[j+n+1]이 된다. (n은 정수 삼각형의 층수이며 array[j] 는 합이 최대가 되는 경로를 통해 도출한 수)

삼각형 가장 아래에서 1칸 윗줄부터 array[j] = array[j] + max(array[j+n],array[j+n+1]) 을 이용하여 최상위에 있는 array[1]에는 정답이 들어가게 된다.




## 💡주의점💡

array[j] = array[j] + max(array[j+n],array[j+n+1])

위의 식을 삼각형 최하위 줄부터 적용하려고 했는데 인덱스 범위를 넘어섰다.
